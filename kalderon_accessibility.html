Our Commitment

AKalderon Arts is committed to making its website accessible to all users, including people with disabilities, in accordance with Israeli Standard IS 5568 — the Israeli equivalent of WCAG 2.1 Level AA — and the Israeli Equal Rights for Persons with Disabilities Law.

Accessibility Features

  • Skip-to-main-content link at the top of every page
  • All images include descriptive alt text
  • Keyboard navigation supported throughout — all interactive elements reachable by Tab
  • ARIA roles and labels on all navigation, lists, forms and regions
  • Sufficient colour contrast ratios (minimum 4.5:1 for normal text)
  • Slideshows and carousels include pause/play controls and keyboard navigation
  • Focus indicators visible on all interactive elements
  • No content flashes more than 3 times per second
  • Page titles are descriptive and unique
  • Language declared in HTML for screen reader compatibility
  • Forms include visible labels and error messages

Known Limitations

Some historical artwork images may have limited alternative text descriptions due to the nature of abstract and non-representational art. We continue to improve descriptions for all works. Some older PDF documents linked from the site may not be fully accessible — we are working to remediate these.

Compatibility

This website is designed to be compatible with:

  • Screen readers: NVDA (Windows), JAWS (Windows), VoiceOver (macOS / iOS), TalkBack (Android)
  • Browsers: Chrome, Firefox, Safari, Edge (latest two major versions)
  • Operating systems: Windows, macOS, iOS, Android
  • Keyboard-only navigation

Technical Standards

This website uses semantic HTML5, WAI-ARIA 1.2 attributes, and follows the Web Content Accessibility Guidelines (WCAG) 2.1 at Level AA, as required by Israeli Standard IS 5568.
